Vite არის თანამედროვე, ძალიან სწრაფი Front-End build tool და development server.
ის იყენებს ES Modules-ს და სტარტდება თითქმის მყისიერად. აქვს სწრაფი HMR და მარტივი კონფიგურაცია.

Babel არის JavaScript ტრანსპილერი, რომელიც ახალ ES6+ სინტაქსს გარდაქმნის ძველ, ბრაუზერებთან თავსებად კოდად.

ESLint არის კოდის შემმოწმებელი ინსტრუმენტი (linter) JavaScript და TypeScript პროექტებისთვის.
იგი პოულობს შეცდომებს, ცუდ სტილს, უსარგებლო ცვლადებს და აძლიერებს კოდის ხარისხს.

Node.js არის პლატფორმა, რომელიც გაძლევს JavaScript-ის გაშვებას არა ბრაუზერში, არამედ კომპიუტერზე/სერვერზე.

CLI არის პროგრამასთან ურთიერთობა ტერმინალიდან ბრძანებებით.
მაგალითად:

NVM არის ინსტრუმენტი, რომელიც გაძლევს სხვადასხვა Node.js ვერსიის დაყენებასა და მათ შორის გადართვას.
